## CompetingRisks.jl

The Julia package `CompetingRisks.jl` implements Bayesian nonparametric inference algorithms for competing risks data, as described in the working paper *Hierarchically dependent mixture hazard rates for modelling competing risks*:
- the folder `/src` contains the package source code;
- the folder `/paper` contains the latest draft of the paper and the Julia codes to replicate the results and plots reported therein, as well as datafiles for the clinical applications.
